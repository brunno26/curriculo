# Currículo Online

Este é meu primeiro projeto prático desenvolvido com HTML, CSS e JS.

## 📌 Sobre o projeto
Criei um currículo online com o objetivo de apresentar minhas informações profissionais de forma organizada e acessível pela web.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JS

## 🎯 Objetivo
Praticar estruturação de páginas web, estilização com CSS, aplicação de modo escuro com java script.
